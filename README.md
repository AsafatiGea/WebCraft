Website jasa pembuatan website profesional dengan desain **Corporate Black × Gold × Crimson**.

## 📁 Struktur File

```
webcraft-v2/
├── index.html          ← Beranda
├── about.html          ← Tentang Kami
├── services.html       ← Layanan
├── portfolio.html      ← Portofolio + Evaluasi 1 Bulan
├── process.html        ← Proses Pengerjaan
├── team.html           ← Tim (Yessica, Asafati, Enricco, Julius)
├── contact.html        ← Kontak & Form

└── assets/
```

## ✨ Fitur JavaScript

| Fitur | Keterangan |
|---|---|
| Page Loader | Animasi loading saat pertama buka halaman |
| Custom Cursor | Kursor emas kustom (desktop only) |
| Scroll Progress Bar | Bar merah-emas di atas halaman |
| Navbar Scroll | Navbar berubah saat di-scroll |
| Active Nav Link | Link aktif otomatis berdasarkan halaman |
| Hamburger Menu | Menu mobile dengan animasi |
| Scroll Reveal | Animasi muncul saat elemen terlihat |
| Counter Animasi | Angka statistik naik secara animasi |
| Typed Text | Efek ketik otomatis (jika diaktifkan) |
| Form Validation | Validasi form kontak dengan pesan error |
| Back to Top | Tombol kembali ke atas |
| Parallax | Efek parallax halus di hero section |
| Ticker | Teks berjalan otomatis |

## 🚀 Cara Deploy ke Domain

1. **Beli domain** di Niagahoster, Domainesia, atau Hostinger
2. **Upload semua file** via cPanel File Manager atau FTP
3. **Pastikan** semua file ada di folder `public_html`
4. **Akses** via nama domain Anda

## 📸 Cara Tambah Foto Profil Tim

1. Siapkan foto format `.jpg` ukuran minimal 200×200px
2. Rename: `yessica.jpg`, `asafati.jpg`, `enricco.jpg`, `julius.jpg`
3. Taruh di folder `assets/images/`
4. Buka `team.html`, cari komentar di setiap kartu
5. Hapus `<span class="team-initials">` dan uncomment tag `<img>`
